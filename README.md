## 🧠 Data Structures & Algorithms Covered

### 1️⃣ Fundamental Arrays & Pointers

- **Pass by Value vs Pass by Reference**  
  - Demonstrates memory address manipulation and parameter passing
- **2D Matrix Operations**  
  - Matrix transpose  
  - Row-wise calculations
- **Dynamic Memory Allocation**
  - 1D arrays using `new` and `delete`
  - 2D arrays using double pointers

---

### 2️⃣ Sorting & Searching Algorithms

- **Bubble Sort**
  - Ascending Order
  - Descending Order
- **Insertion Sort**
  - Ascending Order
  - Descending Order
- **Merge Sort**
  - Recursive divide-and-conquer implementation
- **Binary Search**
  - Efficient logarithmic searching on sorted arrays

---

### 3️⃣ Linear Data Structures

#### 🔹 Stack (LIFO – Last In First Out)

- Array-based Stack
  - Static implementation
  - Dynamic implementation
- Linked List-based Stack
  - Fully dynamic memory allocation

#### 🔹 Queue (FIFO – First In First Out)

- Array-based Circular Queue
- Linked List-based Queue

---

### 4️⃣ Doubly Linked List (DLL)

📚 **Library Management System Example**

- Manages Book IDs using a Doubly Linked List
- Supports:
  - Insert at Head
  - Insert at Tail
  - Delete Specific Value
  - Remove from Head
  - Remove from Tail
  - Forward Traversal
  - Reverse Traversal
